# Skin Tyee website — visual walkthrough

Full-page screenshots of every page on the **skintyee.ca** WordPress site
(captured from the local stack at `http://localhost:8080`). Individual
news/announcement *posts* are excluded — these are the site's pages.

> Regenerate with: bring the site up (`./website/recover.sh`) then run the
> screenshot script (see `docs/wordpress-runbook.md`). Images live in
