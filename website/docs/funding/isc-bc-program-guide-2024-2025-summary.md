# ISC BC Region 2024-2025 Program Guide — Summary

Source: Indigenous Services Canada (ISC) — British Columbia Region, *2024-2025 Program
Guide* (189 pp). Original PDF: `website/web/public/docs/2024-2025-bc-region-program-guide.pdf`.
Online: <https://afoabc.org/publications-2/>.

This is a community-facing summary produced from the guide (one subagent per chapter).
It is the raw material for the **Funding Information** feature: each program below maps
to a Skin Tyee program area and carries its **Application (PAW)** and **Reporting (DCI)**
requirements + due dates.

> **Key concepts** — **PAW** = Proposal / Application / Work plan (how you apply).
> **DCI** = Data Collection Instrument (the report you owe, with a due date).
> **FNIIP** = First Nations Infrastructure Investment Plan — most capital/housing
> projects must be listed in your FNIIP to be eligible. **GCIMS** = the federal system
> that tracks submissions and can auto-halt funds for overdue reports.

## Program-area mapping (Skin Tyee → ISC chapters)

| Skin Tyee program area | Relevant ISC programs |
|---|---|
| **Housing** | Housing Support Program (HSP), Ministerial Loan Guarantee, CFMP, FNIIP, E-ACRS |
| **Education** | BCTEA, Public/Independent Tuition, PSSSP/UCEPP, PSPP, Student Accommodation, allowances |
| **Lands & Economic Development** | LEDSP (Core/Targeted), CORP, RLEMP, FNLM, Contaminated Sites, FNCIDA |
| **Social Development** | Income Assistance (+ sub-benefits), Assisted Living, Family Violence Prevention |
| **Child & Family Services** | FNCFS/ICFSA, Jordan's Principle, CFS Reform — Jurisdiction & Engagement |
| **Health** | (ISC health largely via FNHA in BC; see Social/CFS + Jordan's Principle health supports) |
| *(cross-cutting)* | Community Development (BSF, EB, TCF, P&ID), Emergency Management (EMAP), Funding Information |
